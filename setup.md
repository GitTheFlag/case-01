# Setup Guide

## Requirements
- Node.js 18+
- git 2.30+

## Installation

```bash
git clone https://github.com/GitTheFlag/case-01
cd case-01
npm install
```

## Environment variables

Create a `.env` file:
```
APP_NAME=devteam
DEBUG=false
SECRET_FLAG=GTF{d1ff_sh0ws_ch4ng3s}
```

Note: never commit your `.env` file.
